# {システム名}：品質保証

## 1. 概要

- 本書は {システム名} の品質保証記録を蓄積し、何が・どこまで保証されているかを管理する。
- 関連ドキュメント：要件 / 情報定義 / 機能仕様 / API仕様

## 2. 共通定義

- ID採番：`QA-{連番}`（不変）
- テスト種別：単体 / 結合 / 受け入れ / 非機能 / 回帰 / 探索的
- 観点：正常系 / 異常系 / 境界値 / 整合性 / 状態遷移 / 性能 / セキュリティ / 可用性
- 結果：合格 / 不合格 / 保留 / 未実施
